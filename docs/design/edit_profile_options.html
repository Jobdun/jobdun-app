Verdict

#1 Hub-and-Spoke is the recommendation: it solves the back-button data-loss class structurally (small saves per section), the hub doubles as a completeness checklist with amber "missing" flags, and the focused mini-pages are the most gloves-friendly editing surface. #2 Sectioned Scroll+ is the budget path if you want it this week — sticky chips + a change-counting save bar on the existing form. #4's stepper belongs behind the completeness banner as a "finish your profile" mode regardless of which editor wins. Today's shipped discard-guard protects all five.
